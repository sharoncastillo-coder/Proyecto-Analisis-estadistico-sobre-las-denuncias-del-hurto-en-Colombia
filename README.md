# Análisis estadístico sobre las denuncias del hurto en Colombia (2022–2024)

Proyecto de investigación desarrollado en Python que analiza las tendencias, patrones y factores asociados a las denuncias de hurto en Colombia durante el periodo 2022–2024.
Se aplican técnicas de estadística descriptiva, análisis de variación, coeficiente de variación, correlación de Pearson, índices de autocorrelación espacial (Moran Global y Local) y matrices de transición, con el fin de identificar comportamientos espaciales y temporales del fenómeno delictivo.
# Objetivo
Responder la pregunta: ¿Qué tendencias, patrones y factores asociados se pueden identificar a partir del análisis de las denuncias de hurtos ocurridos en Colombia entre 2022 y 2024?
El análisis busca aportar evidencia cuantitativa que permita comprender la distribución espacial y temporal del delito y apoyar estrategias basadas en datos.
# Metodología
El análisis incorpora las siguientes herramientas:
Estadística descriptiva
Medidas de variación
Coeficiente de variación
Correlación de Pearson
Índice de Moran Global
Índice de Moran Local (LISA)
Matrices de transición espacial
Análisis exploratorio de datos espaciales (ESDA)
# Instalación
El proyecto se desarrolla en Python y se ejecuta en Google Colab.
Clona el repositorio desde GitHub.
Abre el notebook en Colab.
Monta Google Drive para acceder a los datos.
Actualiza las rutas de los archivos según la ubicación en Drive.
Ejecuta las celdas en orden.
